# Player Role Legal Basis

本文件记录玩家职业设定的现实语境和权限边界。目的不是提供法律意见，而是让正式版的开场、走访、资料解锁和文案更像中国本土制度环境。

## 结论

中国现实中不宜把玩家称为“独立案卷调查员”。这不是常见的正式职业名，也容易让人联想到私家侦探或刑事侦查。

正式版推荐称呼：

- 对外称呼：宗世昌家族信托关系核验项目资料核验员。
- 职业归属：江南省明证档案与法律服务中心项目人员。
- 项目关系：受云山县公证处临时委托，在承办公证员授权范围内协助整理材料。
- 口语称呼：公证处请来帮忙核材料的人。

## 他平时在哪里上班

玩家不是公证处正式在编人员，也不是公证员。更可信的设定是：

> 玩家平时供职于“江南省明证档案与法律服务中心”，这是一家承接档案整理、非诉材料核验、法律文书辅助和政务窗口材料外包的专业服务机构。宗世昌案发生后，云山县公证处通过临时项目委托，将玩家派驻到公证处一个临时案卷工位，协助承办公证员整理资料、建立人物关系索引、跑窗口回执和形成关系核验备忘。

这样玩家有三个身份层：

| 层级 | 身份 | 用途 |
| --- | --- | --- |
| 雇佣单位 | 江南省明证档案与法律服务中心项目人员 | 解释玩家平时在哪上班、为什么懂档案和材料整理。 |
| 项目委托 | 云山县公证处临时聘请的外部核验人员 | 解释玩家为什么能拿委托材料走访窗口。 |
| 案内职能 | 宗世昌家族信托关系核验项目资料核验员 | 解释玩家在本案中具体做什么。 |

这个身份不等于现实中有一个通用职业叫“案卷核验顾问”。它是为了游戏合理性合成出来的项目岗位，但组成部分来自现实中常见的非诉材料整理、公证辅助、档案服务、法律服务外包等工作。

## 学历与工作经历

推荐把玩家写成 28-35 岁之间的非诉材料核验从业者，专业能力足够支撑玩法，但不要强到像万能侦探。

| 项目 | 设定 |
| --- | --- |
| 年龄 | 31 岁左右。足够有项目经验，但还不是能单独拍板的资深法律人。 |
| 本科 | 档案学、信息资源管理、法学或社会工作均可；推荐“档案学本科 + 法学辅修”。 |
| 研究生/证书 | 可设为法律硕士在读/非全，或通过法律职业资格客观题但未执业；不建议直接设成执业律师。 |
| 第一段工作 | 省城档案数字化公司或区县政务服务外包项目，做旧档扫描、目录校对、 OCR 修订、卷宗编目。 |
| 第二段工作 | 江南省明证档案与法律服务中心，参与公证材料预审、亲属关系材料核验、继承类材料清单整理。 |
| 相关经验 | 做过烈属档案补证、房产继承材料整理、涉侨亲属关系证明、旧户籍迁移线索核对。 |
| 弱点 | 没有侦查权、不是公证员、不能决定继承资格；遇到病历、学籍、DNA 等隐私材料必须走授权和正式回函。 |

## 为什么他适合这个案子

- 懂旧档：能读懂县志、知青名册、供销社花名册、旧户籍迁入登记的目录结构。
- 懂材料链：知道单份情绪材料不能定案，必须找出生、户籍、DNA、信托条款等互证。
- 懂窗口流程：知道哪些材料可公开查阅，哪些需要介绍信、授权、回执或承办公证员正式函询。
- 懂县城语境：做过区县项目，知道熟人社会里“有人说”不等于“材料能证明”。
- 不带宗家立场：不是本地宗家人，也不是集团雇员，适合先做相对中立的材料梳理。

## 资格证与证件

玩家不需要、也不应该拥有能“直接调取一切资料”的证件。更可信的是一组普通但有用的职业背景证明。

| 类型 | 建议设定 | 用途 | 边界 |
| --- | --- | --- | --- |
| 身份材料 | 身份证、明证中心工作证、云山县公证处项目派驻函 | 证明“我是谁、受谁委托、来办什么事”。 | 只能说明项目身份，不能替代当事人授权。 |
| 学历背景 | 档案学本科毕业证、法学辅修证明 | 解释玩家能读旧档、编目录、理解亲属关系材料。 | 学历不产生取证权。 |
| 档案能力 | 档案专业人员继续教育证明、档案数字化项目培训证、保密培训记录 | 解释玩家能接触档案整理、扫描目录和旧档索引。 | 不能私自复制涉密、隐私或未开放档案。 |
| 法律基础 | 法律职业资格考试客观题合格记录，或基层法律服务/非诉材料培训证 | 解释玩家懂基本程序、证明力和材料清单。 | 不设为执业律师，不出具法律意见。 |
| 信息处理 | 电子数据/信息安全保密培训、OCR 校对和数据标注项目经历 | 解释资料库检索、档案数字化和材料比对能力。 | 不能黑入系统或绕过授权。 |

推荐组合：

1. 必备：明证中心工作证 + 公证处项目派驻函 + 保密承诺书。
2. 专业背景：档案学本科 + 法学辅修。
3. 加分但不越界：法律职业资格考试客观题合格或非诉材料核验培训证。
4. 不建议：执业律师证、公证员执业证、调查证、警官证、法院调查令持有人身份。

游戏里可出现的随身证件：

- 《云山县公证处宗世昌案资料核验项目派驻函》
- 明证中心工作证
- 保密承诺书回执
- 档案馆临时查阅登记卡

这些证件的作用是打开“窗口咨询、材料登记、回执申请”的门，不是打开“直接看病历、学籍、户籍原件”的门。

## 走访和调资料所需凭证

玩家外出走访时应携带一套“材料包”，不同地点需要不同凭证。这样既贴近现实，也能转化为正式版权限系统。

### 基础随身材料

| 凭证 | 出具方 | 用途 | 游戏作用 |
| --- | --- | --- | --- |
| 身份证 | 玩家本人 | 核验个人身份。 | 所有窗口基础要求。 |
| 明证中心工作证 | 江南省明证档案与法律服务中心 | 证明玩家工作单位和项目人员身份。 | 解释玩家不是社会闲散人员。 |
| 项目派驻函 | 明证中心 + 云山县公证处盖章 | 说明玩家被派驻到宗世昌案资料核验项目。 | 开启公证处、档案馆、社区等基础走访。 |
| 公证处介绍信 | 云山县公证处 | 说明具体走访对象、案由、承办公证员和联系方式。 | 每次走访的核心通行材料。 |
| 保密承诺书回执 | 公证处 / 明证中心 | 承诺不扩散个人隐私和未公开材料。 | 允许接触目录、摘录或脱敏材料。 |

| --- | --- | --- | --- |

| 县一中 | 身份证、介绍信、未成年人信息保护承诺、监护人授权或公证处正式函询 | 学籍事实回函、脱敏家访记录、奖学金撤稿说明 | 未成年人完整学籍档案、成绩和隐私细节 |

### 凭证对应的玩法门槛

| 凭证状态 | 玩家体验 |
| --- | --- |
| 只有工作证 | 可以问路、登记、查公开信息，但大多数窗口不会给案内材料。 |
| 有公证处介绍信 | 可以提交调档申请，得到目录、回执和部分公开/脱敏材料。 |
| 有正式函询 | 可以触发医院、学校、档案馆等单位的正式回复。 |
| 有当事人/监护人授权 | 可以解锁更具体的个人材料，但仍需脱敏展示。 |
| 有专业鉴定委托 | 可以解锁 DNA、血样等后段强证据，由专业机构出具结果。 |

### 文案原则

- 窗口人员不应因为玩家一句“我是调查员”就给材料。
- 每份隐私材料都要有来源路径：介绍信、函询、授权、回执或鉴定委托。
- 被拒绝也应有玩法价值：拒绝回执可以提示缺少哪种凭证，推动玩家回公证处补材料。
- 正式版地点界面可以显示“当前凭证”和“可申请材料”，让权限变化变成可理解的进度。

## 调查辛苦感与成就感

凭证系统的目的不是制造繁琐手续，而是让线索有重量。玩家找到一份关键材料时，应该能回想起它是怎么来的：先看到一个模糊名字，再查目录，再补介绍信，再等回执，最后拿到一份脱敏摘录。这种来路会让破解关系谜题更有成就感。

| 阻力 | 不好的用法 | 好的用法 |
| --- | --- | --- |
| 排队和值班 | 让玩家无意义等待。 | 用值班表改变当天可办事项，迫使玩家选择先查哪条线。 |
| 缺少凭证 | 只弹“无法访问”。 | 明确提示缺少介绍信、函询、授权或鉴定委托，并给出回公证处补材料的任务。 |
| 回执延迟 | 机械倒计时。 | 允许玩家先查旁证，等回执回来时形成“终于对上了”的闭环。 |
| 材料脱敏 | 把信息遮到无法推理。 | 遮掉隐私号码，保留年份、关系、单位、关键词，让玩家仍能推理。 |
| 窗口拒绝 | 纯粹惩罚玩家。 | 拒绝本身成为线索：谁拒绝、为什么拒绝、需要谁签字。 |
| 多单位互证 | 重复跑腿。 | 每个单位给不同证明力：档案给时间，医院给出生/样本，学校给监护，社区给生活连续性。 |

正式版可以把“辛苦感”做成三种反馈：

1. 来路反馈：资料卡显示“来自县档案馆调档回执 / 县一中脱敏回函 / 社区走访记录”。
2. 过程反馈：证据箱显示材料获得路径，例如“匿名信 -> 知青名册 -> 调档申请 -> 旧户籍摘录”。
3. 成就反馈：关系核验成功时，不只说“正确”，而是回顾玩家如何排除错误说法、补齐材料链。

设计底线：

- 辛苦感要服务于理解和成就，不要把玩家困在手续迷宫里。
- 每个手续阻力都必须给出下一步方向。
- 关键证据可以难拿，但不能让玩家不知道为什么拿不到。

## 可写进开场的人物小传

> 你，31 岁，档案学本科、法学辅修，供职于江南省明证档案与法律服务中心。过去几年，你做过旧户籍数字化、公证材料预审和继承类亲属关系核验项目，持有明证中心工作证、公证处项目派驻函和保密承诺回执。你不是公证员，也不是律师；你的工作是在正式核实开始前，把散落在报道、档案、窗口回执和当事人材料里的线索整理成一份承办公证员看得懂、问得下去的关系核验备忘。

## 公证处为什么聘请玩家

公证处不是把公证职责外包给玩家，也不是让玩家替公证员作决定。它聘请玩家的合理理由是：本案在短时间内出现大量跨单位、跨年代、跨地域材料，承办公证员需要有人先做资料梳理和线索核验，把“能正式核实的问题”从八卦、公开报道和旧档索引里分出来。

| 原因 | 案内表现 | 玩家实际做的事 |
| --- | --- | --- |
| 材料量异常大 | 宗家提交公开家谱、集团官网、股权材料；匿名信又指向旧知青、户籍、医院、学校和社区。 | 建资料索引，标注来源、年份、人物和证明力。 |
| 单方材料不可靠 | 宗家急于压下“第七支”传闻，集团材料可能服务于内部口径。 | 将宗家材料与公开档案、社区记录、学校材料互相校对。 |
| 时间窗口紧 | 2020 年 9 月 28 日接案，马上进入国庆中秋假期，窗口单位值班有限。 | 先跑可开放或可申请的窗口，整理回执和缺口清单。 |
| 县城熟人社会 | 宗家和本地机关、医院、学校都有关系，承办公证员直接出面容易被视为表态。 | 以外聘资料核验员身份做前期材料整理，降低公证处过早站队感。 |
| 专业边界需要分工 | DNA、血样、信托条款和亲属关系证明涉及不同专业。 | 玩家只整理材料和疑点，鉴定、函询、正式核实由公证处或专业机构处理。 |

更现实的聘请路径：

1. 宗家一方申请或咨询公证事项，提交公开家谱和信托材料。
2. 公证处收到匿名信和矛盾线索，认为仅靠当事人提交材料不足。
3. 承办公证员决定先建立“关系核验项目”，聘请外部案卷核验顾问整理公开资料、旧档索引和走访备忘。
4. 玩家提交的是《关系核验备忘》；承办公证员再决定哪些材料正式核实、哪些要求当事人补充、哪些送专业机构。

## 现实依据

| 现实制度 | 可借用部分 | 游戏转译 |
| --- | --- | --- |
| 公证审查核实 | 公证机构在办理公证前，需要审查申请事项和证明材料；对有疑义的材料可以核实。 | 承办公证员授权玩家整理资料、跑公开档案、形成核验备忘。 |
| 证明材料清单和政务信息 | 公证事项常围绕身份、亲属关系、出生、死亡、婚姻、档案等证明材料展开。 | 资料库中出现户籍、出生记录、死亡资料、信托文件、单位证明。 |
| 专业机构/人员协助 | 对鉴定、检测、翻译等专业问题，可以委托专业机构或人员处理。 | DNA、血样、翻译信托条款等由专业机构完成，玩家只整理和绑定结果。 |
| 律师调查 / 调查令 | 律师和诉讼程序中存在调查取证、调查令等路径，但不等于普通人有侦查权。 | 若正式版引入律师角色，应作为另一路法律协作，不由玩家直接替代。 |

## 玩家能做什么

- 持公证处介绍信或委托材料，到档案馆、社区、学校、医院窗口咨询可公开或可申请材料。
- 整理当事人提交材料、公开报道、官网公告、论坛线索和旧档索引。
- 做访谈记录、摘录档案目录、形成关系核验备忘。
- 把材料按证明力分类：强证据、辅助印证、关键词入口、低可信传闻。
- 将疑点提交给承办公证员，由公证处决定是否正式核实、函询、委托鉴定或要求当事人补充材料。

## 玩家不能做什么

- 不能拥有刑事侦查权。
- 不能私自调取病历、学籍、户籍、DNA、个人通信和银行流水。
- 不能跟踪、偷拍、窃听、黑入账号或购买个人信息。
- 不能冒充公证员、警察、律师或法院人员。
- 不能单独决定继承资格；玩家提交的是核验报告草案或资料备忘，正式意见由公证机构和后续法律程序处理。

## 开场文案建议

> 你是江南省明证档案与法律服务中心的项目人员。云山县公证处在宗世昌家族信托受益人范围咨询中收到匿名线索，因材料横跨旧档案、医院、学校、社区和集团内部文件，临时委托你进驻公证处项目工位，协助承办公证员整理公开资料、走访可申请单位、核对亲属关系线索，并在假期结束前提交一份关系核验备忘。

## 玩法影响

| 系统 | 调整方向 |
| --- | --- |
| 走访 | 每个地点应显示“可直接查阅 / 需介绍信 / 需当事人授权 / 需专业机构”的材料边界。 |
| 资料库 | 后段隐私材料不能靠搜索直接出现，应由走访、授权、回执或专业检测解锁。 |
| 证据箱 | 玩家绑定的是材料和证明力，不是“侦查结论”。 |
| 最终提交 | 文案改成“关系核验备忘 / 公证报告草案”，避免像法院判决或侦查结论。 |
| 时间系统 | 节假日、值班、回函、盖章和补材料构成现实阻力。 |

## 参考来源

- 中国政府网《公证程序规则》：公证机构在审查中可对有疑义事项和证明材料进行核实，也可委托异地公证机构代为核实。
- 中国政府网《公证程序规则》：涉及鉴定、检验检测、翻译等专业事项时，可由当事人委托或经同意由公证机构代办。
- 司法部政务服务指南“公证审查”：公证处在出具公证前，会对申请事项及证明材料进行调查核实。
- 中国政府网《中华人民共和国公证法》：公证机构对申请公证事项和证明材料有疑义时，应当进行核实或委托异地公证机构代为核实，有关单位或个人依法协助。

设计判断：玩家职业应借用“公证核实和资料整理”的现实质感，而不是借用“私家侦探”的戏剧套路。
