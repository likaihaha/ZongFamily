# Player Motivation Arc

本文件记录玩家在游戏中的调查动机、动力变化和系统反馈方式。它用于正式版开场、备忘录、任务压力、结尾报告和叙事节奏设计。

## 设计原则

- 不把玩家写成宗家亲属、私家复仇者、私家侦探或天选侦探。玩家的身份应保持可投射：江南省明证档案与法律服务中心项目人员，受云山县公证处临时委托，担任宗世昌家族信托关系核验项目资料核验员。这个称呼是游戏设定中的项目角色，不是现实中的正式法定职业名。
- 开局动机必须具体：谁委托、为什么现在查、查到什么程度算完成、有什么时间压力。
- 中段动力必须来自玩家自己的发现，而不是 NPC 告诉玩家“这里很重要”。
- 后段动力应从“完成委托”转为“提交一份能保护事实和人的报告”。
- 玩家可以保持职业冷静，但系统要让玩家感到：报告不是答题卡，它会改变几个人在宗家、学校、社区和信托里的位置。

## 开局动机

| 动机层 | 内容 | 游戏表现 |
| --- | --- | --- |
| 职业委托 | 云山县公证处因宗世昌去世、家族信托咨询和匿名信线索，通过临时项目委托玩家所在机构，派玩家协助核验直系血亲范围。 | 开场委托函、派驻函、案卷编号、提交期限、可走访单位清单。 |
| 程序压力 | 2020 年国庆中秋假期将至，机关值班有限，宗家要求尽快形成书面意见。 | 时间表、今日可办事项、节前/假期/节后窗口。 |
| 独立性压力 | 宗家内部有人希望快速压下传闻，公证处不能只收宗家单方材料。 | 公证处接待笔录、集团舆情草案、匿名信。 |
| 初始疑点 | 公开家谱很完整，但匿名信说“仍有后代在云山”；两者同时成立的可能性不明。 | 开局家谱只显示公开宗家和一个未核验缺口。 |

一句话开局动力：玩家不是主动卷入豪门八卦，而是受公证处委托，在有限时间内为一份可能影响信托资格的关系核验备忘找到可证据化答案。

## 为什么会聘请玩家

云山县公证处聘请玩家的原因，不是让玩家替公证员作决定，而是让玩家承担前期资料核验和案卷整理工作：

- 玩家平时供职于档案与法律服务机构，熟悉旧档索引、材料编目、窗口回执和非诉材料核验。
- 本案材料量远超普通亲属关系核验，涉及公开家谱、集团历程、股权底稿、旧户籍、医院、学校、社区和匿名线索。
- 宗家提供的材料具有单方性，公证处需要先把可正式核实的问题整理出来。
- 接案时间临近国庆中秋假期，窗口单位值班有限，需要有人先跑可开放资料和回执。
- 云山县是熟人社会，公证处直接频繁出面容易被理解为站队；外部核验顾问更适合先做材料索引和走访备忘。
- 玩家提交的是关系核验备忘，不是公证结论；承办公证员仍负责正式核实、要求补充材料、委托鉴定和出具意见。

## 动力变化曲线

| 阶段 | 玩家心理 | 推动材料 | 系统反馈 |
| --- | --- | --- | --- |
| 接案：这是一份工作 | 我需要弄清委托范围，按程序完成报告。 | 委托函、公开专访、集团官网、讣告。 | 备忘录给出简单任务：确认公开家庭、阅读匿名信。 |
| 起疑：公开说法不完整 | 公开家谱看似顺，但匿名信和旧资料有缝。 | 匿名信、临终录音、家族会议纪要、旧照片背注。 | 家谱出现未知分支占位；搜索词从“宗世昌”扩到“知青”“建宁”。 |
| 主动：这是我顺出的线 | 我不是被喂答案，而是自己从资料里找到下一步。 | 知青名册、供销社花名册、调档登记、旧户籍。 | 新地点、新资料、新人物逐步解锁。 |
| 介入：这不只是继承争议 | 被抹掉的人有具体生活：罗月珍离开、罗建宁沉默、陈静返乡、陈嘉东读书。 | 未寄出信稿、明信片、社区走访、家访记录。 | 证据箱里出现情绪材料，但标明不能单独定案。 |
| 压力：结论会伤人 | 宗家会反应，未成年人信息要保护，证人怕担责。 | 舆情草案、学校撤稿、司法咨询、法务意见。 | 最终提交前提示证据充分性、隐私和报告措辞。 |
| 收束：给事实一个位置 | 我不是替谁争产，而是提交一份经得起质询的事实链。 | DNA、医院血样、信托条款、出生记录。 | 公证报告式结局，回顾证据链和排除路径。 |

## 动机不应如何写

- 不要写成玩家为了金钱奖励而查案。奖励可以存在，但不能成为核心动力。
- 不要写成玩家天然站在陈嘉东或宗家任何一边。玩家应站在证据和程序这边。
- 不要让 NPC 一上来告诉玩家“真相很黑暗”。应让玩家先看到正常文件，再自己读出不正常。
- 不要把未成年人陈嘉东过早推到玩家面前。她应是后段动力升级的重要节点。
- 不要把家族成员都写成反派。阻拦调查的人也有自保逻辑、体面压力和现实利害。

## 调查过程中的动力反馈

| 反馈类型 | 触发点 | 玩家感受 |
| --- | --- | --- |
| 资料反馈 | 读到互相矛盾的资料 | “我需要判断来源，而不是照单全收。” |
| 家谱反馈 | 未知节点实名或关系线确认 | “我改变了这张图。” |
| 证据反馈 | 错误传闻被强证据排除 | “我不是猜中答案，是排除了不成立的说法。” |
| 时间反馈 | 节假日导致单位值班、回执延迟 | “调查不是无限制搜索，有现实阻力。” |
| 人物反馈 | 证人、学校、社区表现出顾虑 | “资料背后有人，不只是文档。” |
| 报告反馈 | 最终提交要求证据链和措辞 | “我的结论会被别人拿去使用。” |

## 结尾动力

最终报告不应给玩家“破案爽感”就结束。更好的收束是：

1. 事实链成立：罗建宁、陈静、陈嘉东这一支被放回宗世昌直系血亲图谱。
2. 程序结论克制：报告只确认资格审查所需事实，不替任何人做情绪审判。
3. 人物余波存在：宗家、陈静、陈嘉东、证人都会受到影响，但游戏不把所有后果一次讲死。
4. 玩家获得的不是“赢了宗家”，而是“这份被压住的关系终于有了可被质询的证据位置”。
