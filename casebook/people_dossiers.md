# People Dossiers

本文件记录正式版人物深档。它比 `people_biographies.md` 更细，用于剧情写作、资料正文、访谈口吻、美术参考和关系系统设计。

## 使用规则

- 本文件可以记录“设定存在但游戏未必开局出现”的人物。
- 血脉图只展示玩家已确认或正在调查的关键关系；姻亲、配偶、证人、集团同事应优先进入人物档案池。
- 每个重要人物至少需要：成长经历、公开身份、外貌气质、性格特征、核心欲望、恐惧、利害关系、与主案关系。
- 游戏内不要直接写“误导线”“工具人”“关卡功能”等设计词；这些词只能留在开发案卷中。

## 关键人物深档

### 宗世昌

- 成长经历：出生在云山山地乡镇，少年经历粮站、供销社和县城商贸体系，擅长在人情、物资和政策之间找空间。改革开放后从小商店做起，靠仓储、客运、批发和地产边角项目积累家业。
- 外貌气质：晚年清瘦，头发稀白，穿深色中山装或老式夹克；说话慢，但眼神很硬。
- 性格特征：强控制欲，重体面，擅长把亏欠包装成安排；真正软弱处是旧情和血脉。
- 核心欲望：家业不断、宗家不散、晚年名声完整。
- 恐惧：旧事被公开后，自己多年塑造的地方贤达形象崩塌。
- 利害关系：公开子女需要他的名望和资产秩序；隐藏血脉需要他的血缘承认；集团需要他的死亡不引发不可控审查。

### 李桂兰

- 成长经历：县城手工业家庭出身，早年和宗世昌一起守店、记账、跑货。她不是被动的“正妻”，而是宗家创业期的实际支撑者。
- 外貌气质：照片里常穿素色衬衫，发髻整齐，神情克制；晚年偏瘦，手部有长期劳作痕迹。
- 性格特征：隐忍、精明、爱面子，对子女强硬，对宗世昌旧事选择压住。
- 核心欲望：保住公开家庭的完整感。
- 恐惧：自己一生的付出被一句“另有血脉”轻易抹掉。
- 利害关系：她的沉默既保护了公开子女，也让罗月珍一支长期失语。

### 宗建国

- 成长经历：长子，少年时见过家里最穷和最乱的阶段，成年后走体制路线，习惯用程序和文件说话。
- 外貌气质：方脸，戴老花镜，衣着朴素，像退休干部。
- 性格特征：谨慎、保守、怕事，家族会议上倾向“先别闹大”。
- 核心欲望：让宗世昌后事按公开程序收束。
- 利害关系：不一定最贪财，但害怕全家被旧案拖入舆论。

### 刘素珍

- 成长经历：县财政局退休，熟悉预算、票据和报销流程。
- 外貌气质：短发，常戴细边眼镜，衣着干净。
- 性格特征：现实、会算账，私下提醒宗建国不要替别人背锅。
- 利害关系：维护长房稳定，不愿让子女卷入集团内斗。

### 宗明礼 / 王晓芸

- 宗明礼：审计局干部，性格压抑，知道家族企业账目经不起细查；他反感宗建红一支把公司风险推给亲戚。
- 王晓芸：医院财务，懂病案和费用系统边界；可能成为“病危转院账目”侧面的咨询来源。

### 宗明霞 / 蒋越

- 宗明霞：银行风控，敏感于信托、担保和资金来源。
- 蒋越：外地律师，对云山人情不买账，建议长房保留书面意见。

### 宗建芳

- 成长经历：在妇联系统工作多年，擅长调解、帮扶和写材料。她帮助陈静返乡创业，是出于旧同情、人情债，还是受宗世昌暗示，正式版可以保留暧昧。
- 外貌气质：圆脸，短卷发，衣着得体，说话亲切但会避重就轻。
- 性格特征：会照顾人，也会控制叙述；她知道帮扶记录可能被误读。
- 核心欲望：保住自己“帮助困难女性”的公共形象。
- 利害关系：陈静线一旦被查，她最容易被怀疑是陈静母亲或中间遮掩者。

### 赵德胜 / 赵雨晴 / 郭晨

- 赵德胜：供销社旧办公室退休，知道旧仓储点和人员往来，但不愿提。
- 赵雨晴：社会组织从业者，尊敬母亲，也怀疑母亲对陈静过度照顾。
- 郭晨：县融媒体编辑，能接触撤稿、论坛和舆情口径，是公开话语链的入口。

### 宗建民

- 成长经历：物资局出身，后来把旧物资渠道转换为集团物流资源。他不一定在台前，但手里有仓储和运输旧账。
- 外貌气质：身材宽，嗓门大，爱穿夹克，饭局上很有存在感。
- 性格特征：强势、讲兄弟义气，也讲利益边界。
- 核心欲望：保住二房在集团里的实际控制力。
- 利害关系：他两个儿子接连进入董事长序列，任何新增受益人都会削弱二房话语权。

### 彭秀琴

- 成长经历：早年管物流公司账，懂车队、仓库和现金流。
- 外貌气质：精明干练，常拿小本记事。
- 性格特征：护短，不信外人，对宗建红一支长期不满。
- 利害关系：站在宗明辉、宗明军背后维护物流板块。

### 宗明辉 / 吴倩

- 宗明辉：第三任董事长，2016 年春因外部融资尽调和治理结构调整被推到台前。外表温和，实际优柔寡断，想做现代企业管理，却受父辈掣肘。
- 吴倩：投融资经理，重视资本市场口径，担心家族信托争议影响融资。
- 利害关系：宗明辉曾被推上台，又被宗明军替换，兄弟间有暗伤。

### 宗明军 / 韩露

- 宗明军：代理董事长，2020 年 9 月宗世昌病危后临时主持集团口径。年轻、急躁、爱用新媒体和公关压事。
- 韩露：新媒体运营出身，主张快速统一口径、压热搜、删帖。
- 利害关系：他们最怕“第七支血脉”变成公众故事，影响接班合法性。

### 宗建红

- 成长经历：宗家第二代里最像宗世昌的人。她从柜台、采购、招商一路管到集团，认为自己才是把家业做大的人。
- 外貌气质：短发，妆容利落，常穿深色套装；出场压迫感强。
- 性格特征：果断、强硬、擅长定调；对亲情的理解常被权力逻辑覆盖。
- 核心欲望：让集团控制权不被信托和旧事打乱。
- 利害关系：她既是公开家庭代表，也是最有动机压住隐藏血脉的人。

### 李文斌 / 李强 / 林芸 / 李敏 / 杜鹏 / 李天赐

- 李文斌：外贸公司出身，长期在宗建红身后处理外部饭局和项目关系。
- 李强：基金合伙人，外表精英，习惯用尽调语言谈家事；被卷入金融文件。
- 林芸：基金法务，提醒李强不要在信托问题上留下不当表述。
- 李敏：集团客服主管，夹在母亲宗建红和公众传闻之间，因幼子李天赐被八卦牵连而焦虑。
- 杜鹏：地产销售，爱打听内幕，容易把饭局消息传出去。
- 李天赐：婴儿，只有传闻价值，不应承担真实案情功能。

### 宗建平

- 成长经历：公安系统出身，熟悉县城熟人社会的边界：哪些事能查，哪些事最好不要写进纸面。
- 外貌气质：寸头或短发，肩背厚，少笑。
- 性格特征：警惕、护家、厌恶外人翻旧账。
- 核心欲望：把调查控制在“合法但不扩散”的范围内。
- 利害关系：他能影响调档阻力，也能让玩家感到某些门并非技术上打不开，而是人情上不好开。

### 高春梅 / 宗明安 / 张雪 / 宗明慧

- 高春梅：医院行政，知道病房、人情床位和病案室规矩。
- 宗明安：派出所民警，年轻但谨慎，对家族旧事避而不谈。
- 张雪：学校会计，可能间接接触县一中费用、奖学金撤稿。
- 宗明慧：药企销售，熟悉医院渠道，能把病危转院线和药品费用线拉起来。

### 宗建丽

- 成长经历：宗世昌幼女，出生时宗家已经比过去宽裕。她和父亲年龄差大，更像晚年陪伴者，接触过旧信、旧照片和供销系统档案。
- 外貌气质：清瘦，眼下常有疲态，穿着比兄姐低调。
- 性格特征：敏感、犹豫、有同情心，但不敢轻易背叛公开家庭。
- 核心欲望：弄清父亲晚年到底留下了什么话。
- 利害关系：她可能是最愿意让罗月珍一支被看见的人，也是最容易被兄姐施压的人。

### 孙立成 / 孙晓宁

- 孙立成：客运公司调度，和云山至黔中线路有职业交集，跟宗建丽分居后仍偶尔提供线路旧闻。
- 孙晓宁：大学生，接受互联网信息更快，可能帮宗建丽保存扫描件或聊天截图。

### 罗月珍

- 成长经历：贵州山地家庭出身，作为知青到云山后进入供销仓储点。她年轻时要强，离开云山后把旧事压成沉默。
- 外貌气质：旧照片中扎两条辫子或低马尾，眉眼倔，衣服朴素。
- 性格特征：倔强、自尊、怕欠人情，不愿让女儿成为别人茶余饭后的谈资。
- 核心欲望：女儿能安稳长大，不被宗家吞没。
- 利害关系：她的离开造成证据断裂，也使后代长期缺少身份承认。

### 罗建宁

- 成长经历：幼年随母回黔中，在“父亲是谁”的空白里长大。成年后结婚、生女、离婚，选择不回云山追问。
- 外貌气质：眉眼像罗月珍，神情更疲惫；说话慢，不爱直视旧事。
- 性格特征：回避、克制、保护女儿。
- 核心欲望：让陈静不要被上一代的账拖住。
- 利害关系：她是事实链核心，却可能本人不愿成为公开争夺资产的人。

### 陈启山

- 成长经历：黔中矿机厂工人，和罗建宁有一段现实压力很重的婚姻。
- 外貌气质：粗手，沉默，常穿旧工装。
- 性格特征：不多话，传统，认为“能过日子比知道真相重要”。
- 利害关系：作为陈静登记父亲，他解释陈静姓氏来源，但不能否定罗建宁母系血缘。

### 陈静 / 陈明静

- 成长经历：在黔中长大，早年做过服务业和夜场管理，2014 年回云山开唱响 KTV。她既想摆脱贫困，也想靠近母亲不肯说清的地方。
- 外貌气质：卷发或染发，妆容比县城机关人物更鲜明；笑起来热络，沉默时很冷。
- 性格特征：泼辣、会周旋、护女儿，不相信宗家会主动给公平。
- 核心欲望：让陈嘉东有更好的出路，同时不让女儿被宗家当成谈判筹码。
- 利害关系：她是被宗建芳帮扶的人，也是公开宗家最容易拿来做文章的人。

### 赵海

- 成长经历：外地音响设备商，曾与陈静短暂共同生活，后来离开云山。
- 外貌气质：油滑，爱穿亮色夹克。
- 性格特征：逃避责任，嘴上热情，遇事消失。
- 利害关系：解释陈嘉东登记父亲，但不进入继承资格主链。

### 陈嘉东 / 陈小东

- 成长经历：在 KTV 后巷和学校之间长大，成绩好，自尊心强。她知道母亲辛苦，也知道“宗家”两个字会改变别人看她的眼神。
- 外貌气质：学生短发或低马尾，校服洗得发白，书包整齐。
- 性格特征：聪明、敏感、嘴硬，面对成人争夺时会先防御。
- 核心欲望：考出去，别让母亲再求人。
- 利害关系：她本人未必想继承，但她的存在让信托资格无法被公开家庭轻易关上。
