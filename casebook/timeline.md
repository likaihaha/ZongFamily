# Timeline

本文件记录历史时间线和正式版调查日程。历史时间线用于推理，调查日程用于玩法节奏。

## 历史时间线

| 时间 | 事件 | 用途 |
| --- | --- | --- |
| 1932 | 宗世昌出生。 | 全案源头。 |
| 1934 | 李桂兰出生。 | 公开婚姻线。 |
| 1956-1975 | 宗世昌与李桂兰六名公开子女陆续出生。 | 建立公开家庭版本。 |
| 1968 | 罗月珍作为贵州籍知青到云山县。 | 隐藏血脉入口。 |
| 1969 | 宗世昌与罗月珍在供销系统、仓储点、春节合影等旧档中产生交集。 | 支撑罗月珍线。 |
| 1969 冬 | 罗建宁出生。 | 隐藏血脉第一代。 |
| 1970 | 罗月珍携罗建宁迁回黔中。 | 户籍强证据。 |
| 1971 | 黔中汇款、明信片残片出现“建宁”相关信息。 | 辅助印证。 |
| 1988 | 陈静出生，母亲为罗建宁。 | 母系承接。 |
| 1995 | 世昌集团挂牌；县志收录知青名册。 | 公开企业线与旧档入口。 |
| 1998 | 香港家族信托设立。 | 继承资格规则。 |
| 2004 | 陈嘉东出生。 | 现居后代。 |
| 2014-2015 | 陈静返乡，宗建芳帮扶其档案挂靠和 KTV 创业。 | 宗建芳误认线。 |
| 2019 | 陈静与宗世昌血样进行亲缘比对。 | DNA 强证据。 |
| 2020-09 | 宗世昌病危、转院、去世，信托争议爆发。 | 主线调查触发。 |

## 推荐调查日程

正式版主案建议 9 天：2020-09-28 至 2020-10-06。

| 日期 | 阶段 | 主要内容 | 资料开放倾向 |
| --- | --- | --- | --- |
| 09-28 | 接案 | 公证处委托、公开家谱、匿名信、临终录音入口。 | 公开资料、模糊异常。 |
| 09-29 | 公开口径 | 报刊、官网、县情、宗家公开子女。 | 公开家庭、集团历史。 |
| 09-30 | 节前窗口 | 档案馆、医院、学校可提交正式申请。 | 旧档入口、调档登记。 |
| 10-01 | 国庆中秋 | 机关变慢，家族聚会和论坛舆论升温。 | 八卦、匿名电话、家族会议。 |
| 10-02 | 假期噪声 | KTV、社区、论坛、短视频更活跃。 | 陈静误认、地方传闻。 |
| 10-03 | 民间走访 | 老街社区、KTV、证人访谈。 | 生活关系、辅助材料。 |
| 10-04 | 值班回流 | 医院、学校、社区可补材料。 | 学籍、病案、回执。 |
| 10-05 | 强证据回收 | DNA、血样、信托条款和司法咨询进入后段。 | 血缘和资格证据。 |
| 10-06 | 报告压力 | 整理证据链，提交公证报告。 | 最终提交。 |

## 国庆假期设计用途

- 节前适合申请硬证据。
- 假期中机关变慢，民间线索和舆论噪声变多。
- 家族团聚使宗家会议、争吵、匿名电话更自然。
- 2020 年登记、预约、健康码、复工检查解释资料留痕。
- 时间系统应制造调查节奏，不应惩罚玩家阅读速度。
