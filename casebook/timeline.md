# Timeline

本文件记录历史时间线和正式版调查日程。历史时间线用于推理，调查日程用于玩法节奏。

## 历史时间线

| 时间 | 事件 | 用途 |
| --- | --- | --- |
| 1932 | 宗世昌出生，童年经历抗战和战后动荡。 | 全案源头，也解释其物资观、体面观和供销系统底色。 |
| 1934 | 李桂兰出生。 | 公开婚姻线。 |
| 1956-1975 | 宗世昌与李桂兰六名公开子女陆续出生。 | 建立公开家庭版本。 |
| 1968 | 罗月珍作为贵州籍知青到云山县。 | 隐藏血脉入口。 |
| 1969 | 37 岁的宗世昌已与李桂兰育有五名公开子女；他与 20 岁的贵州籍知青罗月珍在供销系统、仓储点、春节合影等旧档中产生交集。 | 支撑罗月珍线，也解释这不是轻飘飘的豪门风流，而是带有年代、单位和权力差的旧账。 |
| 1969 冬 | 罗建宁出生。 | 隐藏血脉第一代。 |
| 1970 | 罗月珍携罗建宁迁回黔中。 | 户籍强证据。 |
| 1971 | 黔中汇款、明信片残片出现“建宁”相关信息。 | 辅助印证。 |
| 1988 | 陈静出生，母亲为罗建宁。 | 母系承接。 |
| 1995 | 县志收录知青名册。 | 旧档入口。 |
| 1997 | 世昌集团完成集团化工商登记并挂牌成立；该“挂牌”不是证券上市。 | 公开企业线入口。 |
| 1998 | 香港家族信托设立。 | 继承资格规则。 |
| 2005 | 世昌集团改制，宗建红接手日常经营；宗世昌退出一线管理但保留重大事项影响力。 | 股权与公开家庭版本。 |
| 2004 | 陈嘉东出生。 | 现居后代。 |
| 2008-2013 | 物流园、冷链配送和建材市场扩张。 | 县域龙头成形。 |
| 2014-2015 | 陈静返乡，宗建芳帮扶其档案挂靠和 KTV 创业。 | 宗建芳误认线。 |
| 2016 | 世昌集团春季治理调整，宗明辉出任董事长；外部融资方要求剥离“创始人特别意见”。 | 融资压力和集团控制权冲突。 |
| 2019 | 陈静与宗世昌血样进行亲缘比对。 | DNA 强证据。 |
| 2020 | 世昌集团年营收约 42 亿元、员工约 4200 人，是云山县县域龙头但未上市。 | 地方豪门尺度。 |
| 2020-09 | 宗世昌病危、转院、去世，信托争议爆发。 | 主线调查触发。 |

## 县志人口口径

| 年代 | 推荐人口口径 | 变化 |
| --- | --- | --- |
| 1960s-1970s | 35-42 万 | 农业和供销系统为主，县城小。 |
| 1980s | 43-50 万 | 商贸恢复，个体户和小批发出现。 |
| 1990s | 户籍 52-56 万 | 县城商贸、运输、学校和建材扩张。 |
| 2000s | 户籍 58-61 万，常住 46-50 万 | 外出务工明显，户籍和常住分离。 |
| 2010s | 常住县城上升，乡镇下降 | 返乡创业、陪读、物流就业增加。 |
| 2020 | 户籍约 62 万，常住约 48 万，县城建成区约 18 万 | 支撑县域龙头，也保留熟人社会。 |

## 国庆假期设计用途

- 节前适合申请硬证据。
- 假期中机关变慢，民间线索和舆论噪声变多。
- 家族团聚使宗家会议、争吵、匿名电话更自然。
- 2020 年登记、预约、健康码、复工检查解释资料留痕。
- 时间系统应制造调查节奏，不应惩罚玩家阅读速度。

## 时间分支与重玩

- 一次调查不应默认看完所有人、所有访谈和所有资料。
- 主线必要事实必须可完成；错过某个证人或窗口时，应有替代证据链。
- 可错过内容主要是人物侧面、情绪材料、支线解释、低可信传闻和不同人的说法。
- 结尾报告应按本次实际取得的材料生成，形成“我的案卷版本”。
- 二周目动力来自换调查顺序：先查学校会更早看到陈嘉东，先查集团会更早理解压制口径，先查旧档会更早进入罗月珍和罗建宁。
