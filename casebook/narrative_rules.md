# Narrative Rules

本文件记录叙事方法论。影视手法可以借鉴，但必须转译成互动行为。

## 总原则

影视提供张力，系统提供参与。

玩家不是被动观看反转，而是通过搜索、阅读、收藏、绑定证据和排除误导亲手完成反转。

## 影视手法转译

| 手法 | 互动转译 | 本案示例 |
| --- | --- | --- |
| 冷开场 | 开局给几份互相不完整的材料，允许玩家自由选择。 | 委托函、匿名信、临终录音、公开讣告。 |
| 信息差 | 资料冲突，玩家用来源和证据链判断。 | 公开六子女口径 vs 第七支传闻。 |
| 红鲱鱼 | 错误方向可调查、可收藏、可绑定，但会被强证据排除。 | 宗建芳、钱树林、李天赐。 |
| 反转 | 关系核验后，家谱节点或证据链改变状态。 | 陈静不是宗建芳女儿。 |
| 每幕结尾留问题 | 完成核验后，备忘录解锁新方向。 | 从公开家庭进入“罗姓女知青”。 |
| 情绪真相 | 情绪材料解释动机，不替代定案证据。 | 李桂兰未寄出信稿、旧明信片。 |

## 写作约束

- 每个重要反转必须有可操作证据链。
- 每条红鲱鱼必须能被玩家通过证据排除。
- 每段情绪材料必须标明用途：定案证据、辅助印证、动机解释或关键词入口。
- 每次出现新问题，都要落到搜索词、走访地点、人物节点或关系卡。
- 重要叙事节点应允许多条路径抵达，避免唯一阅读顺序。

## 玩家动力曲线

| 阶段 | 动力 | 互动反馈 |
| --- | --- | --- |
| 接案 | 职业委托、提交期限和公证程序。 | 委托函、案卷编号、今日可办事项。 |
| 起疑 | 公开家谱与匿名信互相不完全吻合。 | 未核验分支占位、异常关键词。 |
| 主动追查 | 玩家自己从资料中顺出下一步。 | 新地点、新人物、新资料渐进解锁。 |
| 情感介入 | 被抹掉的人有具体生活，不只是继承资格。 | 信稿、明信片、社区走访、家访记录。 |
| 结论压力 | 报告会影响宗家、证人和未成年人。 | 提交前检查证据充分性、隐私和措辞。 |
| 收束 | 给事实一个经得起质询的位置。 | 公证报告式结局和证据链回顾。 |

## 调查辛苦感

- 辛苦感要来自材料取得路径，而不是界面故意难用。
- 凭证、排队、值班、回执、脱敏和窗口拒绝都应给玩家下一步方向。
- 关键资料的获得路径应被记录下来，让玩家在关系核验成功时看到自己如何一步步拼出证据链。
- 失败应推进认知：缺介绍信、缺正式函询、缺授权、缺专业鉴定，都比单纯“不可访问”更有意义。
- 每个单位提供不同证明力，避免重复跑腿：档案给历史时间，医院给出生/样本，学校给监护关系，社区给生活连续性，集团给控制权和信托冲突。

## 开局直觉判断

玩家一开始不应被任务清单牵着鼻子走，而是先通过直觉判断调查方向。直觉来自材料的“不对劲”，再由后续证据链校准。

| 直觉来源 | 开局表现 | 可能引导的方向 |
| --- | --- | --- |
| 语气不对 | 匿名信用词含糊，但反复强调“仍在云山”。 | 先查云山本地近年学校、社区、工商资料。 |
| 时间不对 | 宗世昌去世、信托咨询、国庆假期窗口挤在一起。 | 先查公证处接待、医院转院和集团舆情反应。 |
| 家谱太整齐 | 公开六子女关系完整，却出现未知分支传闻。 | 先核公开家庭，再查旧知青和供销社。 |
| 人物回避 | 宗家材料避谈罗姓、建宁、旧照片。 | 搜索关键词“罗”“建宁”“知青”“供销”。 |
| 来源不稳 | 论坛和八卦有名字，但称谓混乱。 | 把传闻当入口，而不是当证据。 |

设计要求：

- 开局给玩家 2-3 个可凭直觉选择的方向，而不是唯一正确任务。
- 每个方向都应有反馈：有的给强线索，有的给误导入口，有的提示需要凭证。
- 直觉可以错，但错得有价值；错误方向应教会玩家区分来源、时间和证明力。
- 备忘录只记录玩家发现过的异常，不应提前替玩家写出后段答案。

## 多入口同收束

玩家开局选择哪个方向都应能继续推进。不同入口给不同类型的证据和阻力，但最终都应收束到同一组核心节点：罗月珍、罗建宁、陈静、陈嘉东、信托资格。

| 开局方向 | 第一批反馈 | 中段转接 | 最终收束 |
| --- | --- | --- | --- |
| 公开家谱 / 宗世昌 | 公开六子女、集团官网、宗家口径。 | 发现公开口径没有解释匿名信，转向旧知青、信托或家族会议。 | 确认公开家庭只是基准，不排除隐藏血脉。 |
| 旧档 / 知青 | 知青名册、供销社花名册、照片背注。 | 转向罗月珍、罗建宁、旧户籍和接生登记。 | 收束到隐藏血脉第一代。 |
| 学校 / 陈小东 | 家长会帖、奖学金撤稿、学籍门槛。 | 转向陈静、社区、KTV 和监护关系。 | 收束到陈嘉东是陈静之女。 |
| 社区 / KTV | 租赁备案、复工检查、社区走访。 | 转向陈静返乡、宗建芳帮扶、陈嘉东同住。 | 收束到陈静生活线和陈嘉东居住连续性。 |
| 集团 / 股权 | 股权底稿、治理调整、舆情草案。 | 转向信托条款、法务意见、宗家压制口径。 | 收束到公司控制权不能替代血缘资格。 |
| 论坛 / 传闻 | 宗建芳误认、钱树林认父、李天赐传闻。 | 通过强证据排除，保留关键词入口。 | 收束到真正需要证明的母系链。 |

设计规则：

- 任一路线至少在 10-15 分钟内给玩家一个“可继续追”的新关键词或新地点。
- 错误方向不能只结束为“错了”，必须把玩家导回证据链：排除谁、为什么排除、下一步查什么。
- 每个核心节点至少有两条入口路径，避免唯一解锁顺序。
- 最终报告回顾时，应按玩家实际走过的路径组织证据，而不是强行按固定章节复述。

## 首次体验约束

- 开局信息有限，但行动明确。
- 不开局展示完整隐藏血脉链。
- 不让资料库直接吐出后段强证据。
- 教学嵌入备忘录和第一条关系核验，不使用大段弹窗说明。
- 玩家第一次失败必须知道缺的是人物、证据还是调查阶段。
