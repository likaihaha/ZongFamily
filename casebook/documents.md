# Documents

本文件是资料文案库索引。正式版应把资料正文从代码迁移到结构化数据，并为每份资料声明可见性和解锁条件。

## 可见性层级

| 层级 | 含义 |
| --- | --- |
| `public` | 开局可见。 |
| `keyword` | 玩家先获得关键词后可搜到。 |
| `location` | 走访特定地点后可见。 |
| `task` | 完成待办或提交申请后可见。 |
| `chain` | 证据链进入后段后才可见。 |
| `decoy` | 可见但低可信，负责误导或入口。 |
