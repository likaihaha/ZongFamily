《云山宗氏案》真人试玩执行包

用途：记录自动试玩无法判断的体验问题，重点观察走访问询后的“更多回查 / 新问询入口 / 首次展开 / 1 项待回查”是否能被玩家自然注意到。填写后可生成 `manual_playtest_result_YYYYMMDD_observer.md` 格式的回收 Markdown。

执行路线

  1. 打开无历史存档的 `game/index.html`，不使用 `?playtest=1` 或 `?autotest=1`。
  2. 只告诉玩家目标：判断宗世昌信托继承资格中的隐藏血脉。
  3. 观察玩家是否能从资料库、走访页或备忘录找到第一个公开宗家入口。
  4. 让玩家自由推进到世昌集团，办理取件并阅读信托入口材料。
  5. 在世昌集团问询任意一个问题后，不提示下一步，观察侧栏阶段提示和调查备忘录。
  6. 打开调查备忘录，观察“更多回查 / 新问询入口 / 首次展开 / 1 项待回查”是否被注意到。
  7. 继续追踪罗月珍、罗建宁、陈静、陈嘉东链条，记录搜索停顿和是否依赖锁定提示。
  8. 让玩家尝试完成家谱关系核验和最终提交，记录错误证据绑定点。

关键观察点

  • 问询后，玩家是否先看走访结果区的入口材料按钮？
  • 离开走访页后，玩家是否还能从侧栏阶段提示回到先看材料？
  • 玩家打开备忘录时，是否自然注意到“更多回查”区域？
  • “新问询入口 / 首次展开 / 1 项待回查”是否被理解为可行动入口？
  • 玩家是否理解锁定搜索结果是在提示走访方向，而不是搜索失败？
  • 玩家是否在家谱证据绑定中误把传闻材料当强证据？

通过标准

  • 玩家在一次现场问询后，2 分钟内能找到至少一个回查入口。
  • 玩家能说出“先看材料”对应哪份资料，并实际打开至少一份入口材料。
  • 玩家至少能独立推进到罗建宁线，不需要主持人直接给出姓名。
  • 玩家理解锁定结果代表当前资料未入卷，需要走访或阅读前置材料。
  • 玩家在家谱证据绑定中没有明显误用传闻材料。
