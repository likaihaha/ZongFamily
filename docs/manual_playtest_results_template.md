# 真人试玩结果回收模板

更新时间：2026-05-19 07:39 Asia/Shanghai

这份模板用于把 `docs/manual_playtest_packet.html` 的真人试玩记录回收到同一格式。当前目标不是替代真人观察，而是确保试玩后能判断是否需要调整“更多回查 / 新问询入口 / 先看材料 / 锁定提示”。执行包可直接生成一份同结构 Markdown，文件名建议使用 `manual_playtest_result_YYYYMMDD_observer.md`。

## 填写要求

- 每轮试玩复制一份本模板，文件名建议使用 `manual_playtest_result_YYYYMMDD_observer.md`。
- 也可以在 `docs/manual_playtest_packet.html` 填写记录后点击“生成回收 Markdown”，再把生成内容保存为同名结果文件。
- 保留玩家原话，不要只写观察员总结。
- 记录所有超过 30 秒的停顿、主动搜索词和主持人提示。
- 若没有完成到最终提交，也要记录最远推进位置和中断原因。
