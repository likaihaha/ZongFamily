# 真人试玩检查清单

更新时间：2026-05-18 21:30 Asia/Shanghai

这份清单用于补足自动试玩无法判断的体验问题。当前重点不是继续增加 UI 入口，而是确认玩家是否能自然注意到走访问询后的“更多回查 / 新问询入口 / 首次展开 / 1 项待回查”。

## 准备

- 使用无历史存档的浏览器窗口打开 `game/index.html`。
- 不使用 `?playtest=1` 或 `?autotest=1`。
- 试玩者不要提前阅读真相核心、搜索路线复盘或测试报告。
- 记录每次停顿超过 30 秒的位置、玩家主动搜索词、是否点击走访页问询、是否打开调查备忘录。

## 路线

1. 开局只告诉玩家目标：判断宗世昌信托继承资格中的隐藏血脉。
2. 观察玩家是否能从资料库、走访页或备忘录找到第一个公开宗家入口。
3. 让玩家自由调查到世昌集团，办理取件并阅读信托入口材料。
4. 在世昌集团问询任意一个问题后，不提示玩家下一步，观察是否注意到侧栏阶段提示和调查备忘录。
5. 打开调查备忘录后，观察“更多回查”是否被注意到，尤其是“新问询入口”“首次展开”和“1 项待回查”。
6. 继续让玩家追踪罗月珍、罗建宁、陈静、陈嘉东链条，记录每次搜索停顿和是否依赖锁定提示。
7. 让玩家尝试完成家谱关系核验和最终提交，记录错误证据绑定点。

## 观察问题

- 玩家问询后，是否先看走访结果区的入口材料按钮？
- 玩家离开走访页后，是否还能从侧栏阶段提示回到先看材料？
- 玩家打开备忘录时，是否自然注意到“更多回查”区域？
- “新问询入口 / 首次展开 / 1 项待回查”是否表达清楚，还是被理解成普通状态说明？
- 玩家是否能理解被锁定搜索结果是在提示走访方向，而不是搜索失败？
- 玩家是否在家谱证据绑定中误把传闻材料当强证据？

## 通过标准

- 玩家在一次现场问询后，2 分钟内能找到至少一个回查入口：走访结果、侧栏阶段提示或备忘录“更多回查”。
- 玩家能说出“先看材料”对应哪份资料，并实际打开至少一份入口材料。
- 玩家至少能独立推进到罗建宁线，不需要主持人直接给出姓名。
- 玩家理解锁定结果代表“当前资料未入卷，需要走访或阅读前置材料”。
- 家谱页不会因为证据绑定卡片或关系解释造成明显误会。

## 判定

- 如果 2 名试玩者都漏看“更多回查”，优先调整视觉权重或动效时长，不增加新的入口。
- 如果玩家能看到“更多回查”但不知道要点哪里，优先改折叠标题和按钮文案。
- 如果玩家知道要点但不理解“先看材料”，优先把入口材料按钮文案改得更具体。
- 如果玩家主要卡在搜索词，不改备忘录，回到 `docs/search_route_review.md` 补搜索路径。
